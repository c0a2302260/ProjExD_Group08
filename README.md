## ゲームの実装
### 共通基本機能
* musou_kokaton.py
* ビームの速度調整

### 担当追加機能
* 残弾数の追加(担当:渋谷) : 自機が発射するビームの残弾数を管理する機能
* ウェーブ制で敵の追加(担当:臼井) : 敵を倒すとウェーブが上昇し、敵の数が増える機能
* 残機の追加(担当:浜屋) : 自機の残機を表示する機能。0になるとゲームオーバー
* クールダウンのスキル（担当:林田）:スキル使用する場合スコアは減らずクールタイムが発生する
* 背景スクロール（担当：飯野）：背景を右にスクロールする
